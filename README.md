# 🌤️ React Weather App

A responsive weather application built using React and OpenWeather API.

## Features
- ✅ Search weather by city
- ✅ Dynamic background images based on weather conditions
- ✅ Temperature, humidity and wind information
- ✅ Responsive design
- ✅ Error handling
- ✅ Enter key support

## Technologies Used
- React
- JavaScript (ES6+)
- CSS3
- OpenWeather API
- Vite
